Key Deliverables

🌐

Network Design & Deployment

Designed and deployed the corporate network structure across all 12 properties, including VLANs, managed switches, wireless infrastructure, and firewall configurations.

🖥️

Server Virtualization

Deployed VMware ESXi and Hyper-V hypervisors at each site, consolidating physical servers and improving reliability and resource utilization.

🔒

Site-to-Site VPN

Configured Site-to-Site VPN tunnels connecting all hotel properties to the corporate office, enabling centralized management and shared services.

☁️

Cloud Backup (AWS)

Implemented offsite cloud backup to AWS for critical business data, replacing unreliable tape-based backups and ensuring recovery capability across all sites.

📧

Office 365 Migration

Led migration from on-premises email to Office 365, managing DNS changes, user provisioning, and post-migration support across all properties.

🌍

Domain & Web Hosting

Managed corporate domain infrastructure and hotel websites hosted on AWS, including DNS management, SSL certificates, and hosting maintenance.
