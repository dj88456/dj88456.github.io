核心功能

⚡

实时同步

Figma 变量修改后，运行一条命令即可将所有 Token 更新到代码仓库。

🎯

多平台输出

同时生成 CSS Variables、JS/TS 对象、iOS Swift、Android XML 格式。

🔗

CI 集成

提供 GitHub Action，设计稿更新后自动触发 Token 同步并提交 PR。

🌓

主题支持

完整支持 Figma 多模式变量，自动生成亮色/暗色/高对比等多套主题。
