Key Deliverables

🏗️

Subscription Design

Platform/application separation model with dedicated subscriptions for Identity, Connectivity, Management, and workload landing zones.

🌐

Hub-and-Spoke Networking

Central hub with Azure Firewall for traffic inspection, Private DNS zones, and ExpressRoute for hybrid connectivity to on-premises.

🔐

RBAC Framework

Custom role definitions and group-based assignments for platform team, application teams, and read-only auditors. All elevated roles gated via PIM.

📋

Azure Policy Guardrails

Deny policies for non-Canadian regions and public IPs; audit policies for tagging and diagnostic settings; auto-remediation for log forwarding.

🔒

Private Endpoints

All PaaS services (Storage, SQL, Key Vault) accessed via Private Endpoints — no public exposure in production environments.

📊

Governance Documentation

Cloud governance documents including RBAC role definitions, naming conventions, tagging taxonomy, and subscription vending process.
