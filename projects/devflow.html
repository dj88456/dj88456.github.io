DevFlow CLI

一款面向开发者的命令行工具，自动化日常 Git 工作流、一键生成规范的 Changelog，并集成 CI/CD 状态检查，让每次发版都更顺畅。

核心功能

⚡

智能 Commit

遵循 Conventional Commits 规范，自动提示类型与范围，生成标准化提交信息。

📋

Changelog 生成

基于 Git 历史自动生成分类清晰的 CHANGELOG.md，支持自定义模板。

🔍

CI 状态检查

发版前自动拉取 GitHub Actions / CircleCI 构建状态，阻断失败的发布。

🏷

语义化版本

根据提交类型自动推算下一个 semver 版本号，支持 pre-release 标签。

快速开始

# 全局安装 npm install -g devflow-cli # 初始化项目配置 devflow init # 交互式提交 devflow commit # 生成 Changelog 并发布 devflow release --minor
