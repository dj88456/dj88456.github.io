ChatContext

基于 LLM 的文档问答系统。上传 PDF、Markdown 或网页内容，用自然语言提问，系统返回精准答案并附带原文引用，告别信息搜索焦虑。

核心功能

📎

多格式支持

支持 PDF、Markdown、TXT、网页 URL，自动解析并清洗文本内容。

🔍

语义检索

使用向量相似度搜索，比关键词搜索更理解语义，找到真正相关的段落。

📌

引用溯源

每条答案都附带原文段落和页码，方便核实信息来源，杜绝幻觉。

💾

对话历史

支持多轮对话，自动记忆上下文，连续追问不丢失语境。

快速部署

# 克隆项目 git clone https://github.com/yourname/chatcontext cd chatcontext # 配置环境变量 cp .env.example .env # 编辑 .env，填入 OPENAI_API_KEY # 一键启动（Docker） docker compose up -d
